O projeto foi desenvolvido para um usuário que deseja esconder as mensagens compartilhadas com amigos e familia, para realizar um aniversário surpresa de sua mãe, sem que ela veja o que está sendo escrito. 
Foi desenvolvido com dois campos de entrada, um para inserir a mensagem que o usuário deseja escrever e esconder, e a quantidade de deslocamento que essa frase terá, é gerado o código e o usuário copia e cola no primeiro campo para desifrar.


No planejamento defini as variaveis de entrada, no caso string e offset, depois identifiquei o processamento que alteraria o deslocamento, por fim implementei no html ao chamar a função o retorno do código cifrado e decifrado, para aparecer na tela.
